# BiteClip v0.1.2

Small quality-of-life release.

## What's New

- Fade in and fade out are now OFF by default.

## Why

Most users export quick clips without fades, so this avoids accidental fade effects on first use.

## Downloads

- `BiteClip-Setup-0.1.2.exe` - recommended installer
- `BiteClip-Setup-0.1.2.zip` - portable zip build

## Update Behavior

Users already on `v0.1.1` should now be able to detect this release through the in-app update check.

## Windows Warning

This build is still unsigned. Windows SmartScreen may show an "unknown publisher" warning.
